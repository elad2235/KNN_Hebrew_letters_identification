requirements:
    all project requirements located at requirements.txt file

    to install all requirements run the following command:
        pip install -r requirements.txt

    Important:
        when using pycharm as IDE the program won't work outside of pycharm project


run program:
    cd to python script directory

    run the following command:
        python knn_classifier.py ./hhd_dataset


program output:
        confusion_matrix.csv
        results.txt
        program runtime (printed in terminal)
